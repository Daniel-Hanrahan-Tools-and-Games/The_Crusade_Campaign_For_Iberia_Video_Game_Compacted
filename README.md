# The_Crusade_Campaign_For_Iberia_Video_Game_Compacted
Compacted version of The_Crusade_Campaign_For_Iberia_Video_Game for the Tic-80 written in Lua

The executable and source code is in one file.

To load the file you drag the file into Tic-80 or have it loaded into your sd card if you are using raspberry pi bare metal.

To run the game after you loaded the game you type in run then press enter.

To get to source code after you loaded the game you press escape key.

This is mostly a one to one port of The_Crusade_Campaign_For_Iberia_Video_Game with the exception of mod support.
